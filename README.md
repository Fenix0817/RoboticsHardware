**RAM**: 512MB DDR3 (shared with GPU) <br>

**WiFi**: On-board with antenna <br>
**Memory Expansion**: Micro SD Card (Max. 32GB) <br>
**Ports**: 1 USB 2.0 Host, 1 USB OTG, 13 GPIOs <br>

**OS**: Ubuntu, Android <br>
